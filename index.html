RESEARCH

Development of predictive geological models using machine learning to ease hidrocarbon exploration

An analytic investigation which scope is to demonstrate that data science workflows can be used to ease Oil & Gas exploration and its nature as a reliable source of geological insights. We'll construct predictive models based on seismic attributes and petrophysical parameters using supervised machine learning algorithms. The investigation made use of the project OGA MNSH (under Creative Commons 3.0 license) which can be found in the cloud-based portal of Terranubis (dGB Earth Sciences; OpendTect).

Development of a computational tool for reservoir characterization using prestacked seismic

Development of a Python related computational tool for a rapid evaluation of seismic data, computation of AVO attributes and the extraction of AVO anomalies. The code allowed me to validate the gas bearing sands within the Plober Formation, Browse Basin (Australia). The investigation made use of the project NW Shelf Australia - Poseidon 3D (under Creative Commons 3.0 license) which can be found in the cloud-based portal of Terranubis (dGB Earth Sciences; OpendTect).

PROJECTS

PCT

Use of Python v. 3.7 and object oriented programming in the development of the computational tool PCT (Prestack Characterization Tool) for the inspection, visualization and evaluation of pre stacked amplitudes, besides, the execution of standardized workflows in seismic amplitude investigations. Design of four programming modules focalized in the inspection and visualization of seismic data, the computation of AVO attributes (intercept and gradient) by OLS linear regressions and the construction of cross plots for the visualization and extraction of amplitude anomalies.

Status: On going!
